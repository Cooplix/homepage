### Installation

1. To develop in 'develop' branch

```sh
$ git clone https://github.com/Cooplix/homepage.git
```
1. To develop in 'main' branch
```sh
$ git clone https://github.com/Cooplix/homepage.git
$ git checkout -b main
$ git pull origin main
```
